# Bright-TV-Case-Study
This project analyzes streaming platform data for BrightTV, with the goal of uncovering insights into user behavior, content performance, and business growth opportunities. Using SQL, Excel, and data visualization techniques, the case study focuses on transforming raw data into meaningful recommendations for decision-making.
SQL for Data cleaning, querying, and analysis ,
 Excel for pivot tables, charts ,
GitHub for version control and project documentation.
